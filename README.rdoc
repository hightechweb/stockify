== README

This is the repository for the Stockify RoR app 

This app includes the following:

* Authentication system

* User stock ticker tracking

* User search stock tickers using API

* User can add stock from search results to portfolio

* User can lookup friends, or other users by name or email

* User can view stock portfolio of self or friends for investing ideas

* Built in bootstap for mobile responsive

* Deployment instructions

* ...


Please feel free to contribute!
